# 简介

这是flask应用模板。

# 功能

1. basic http登录。
2. 微信小程序登录、微信公众号登录、微信支付。
3. 文章管理（实例功能，暂时没有更新）。

# 第三方

1. flask
2. flask-sqlalchemy
3. flask-restx
4. Flask-Bootstrap
5. marshmallow
